# Architecture

## Overview

This is a documentation-only repository — no code, no build system. It archives the Volere Requirements Specification Template and supporting materials.

## Structure

```
volere-requirement/
├── a–c    Templates (PDF + DOC)
├── d–g    Case studies (PDF)
└── h–i    Atomic requirement spreadsheets (XLS)
```

## Design Decisions

- **Preserve original filenames** — the letter prefixes match the original package ordering and are referenced in the package contents PDF.
- **No format conversion** — files are kept in their original formats (.doc, .pdf, .xls) to preserve fidelity.
