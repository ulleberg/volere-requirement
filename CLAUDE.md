# Volere Requirements Template

## What This Is

A reference archive of the Volere Requirements Specification Template (v15) and supporting case studies. This is documentation, not code — the repo preserves the original materials for reference.

## File Naming

Files are prefixed with letters (a–i) reflecting the original package ordering. Preserve this convention.

## Key Files

- `b Volere Template15.doc` — the editable template, use this as a starting point for new projects
- `h/i` XLS files — atomic requirement card format, useful for individual requirement tracking
